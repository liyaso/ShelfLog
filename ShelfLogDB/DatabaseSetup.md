# Setup Guide
To create the databases run the scripts in this order:
1. Schema
   1. create_database.sql
   2. create_tables.sql
   3. create_views.sql
  
2. Inital Data
   1. insert_users.sql
   2. insert_authors.sql
   3. insert_books.sql
   4. link_books_authors.sql
   5. insert_sample_data.sql
  
In db_config.js, make sure you change the password to your password.
