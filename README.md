# ShelfLog
Our application, ShelfLog, is a Reading Tracker designed to help users organize their reading life, track progress, set goals, and gain insights into their reading habits. The system supports all required basic database functions and introduces a technically challenging advanced feature that goes beyond what typical reading-tracking applications offer. Other book apps, such as Goodreads work similarly to our application. However, Goodreads focuses more on reviews and basic tracking. ShelfLog offers a Behavior Analytics Engine that not only helps users gain insights on their progress, but also helps users learn more about their reading habits. Our data will include user management, books, reading lists created by users, user goals and progress, and reviews. We’ll be getting our book data from a free API so our users have a variety of books at their disposal.
